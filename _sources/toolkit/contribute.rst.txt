How to contribute
=================

We are currently setting up the code review and build infrastructure for the
project.

For now, you may submit bugs as issues in the `CLIP OS bug repository on GitHub
<https://github.com/CLIPOS/bugs>`_ and patches as pull requests in each
repositories.

If you have specific questions, please send a mail to `clipos@ssi.gouv.fr
<mailto:clipos@ssi.gouv.fr>`_.
